#### 2.1 Product Development

This section covers the product development process for developers on the DMP platform.

**Navigation Path:** `[Products] – [Product Development]`

The product development module allows developers to:
- Create and manage hardware products
- Configure product functions and features
- Set up device interaction capabilities
- Manage firmware and OTA upgrades
