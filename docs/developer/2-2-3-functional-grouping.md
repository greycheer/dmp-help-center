#### 2.2.3 Functional Grouping

**Functional Grouping** is used to organize device functions into a tree-structured configuration for App device settings.

**Key Concepts:**
- **Function Groups** control the configurable settings section in the App
- Functions can be organized hierarchically
- Only **Bool** and **Enum** type attribute functions support App-side control
- Other function types are display-only

**Configuration Steps:**

1. Navigate to **Product → Product Development**
2. Click **Continue Development** for the target product
3. Select the **Device Interaction** subpage
4. Click **Functional Grouping** to configure

**Note:** Built-in App features such as product basic information, device reboot, device sharing, and device information are not controlled by Function Groups.
