# DMP Help Center

Welcome to the **DMP (Device Management Platform) Help Center**. This documentation provides comprehensive guides for both administrators and developers working with the DMP platform.

## 📋 Documentation Overview

### 📘 Admin Manual
For platform owners and senior administrators who manage the DMP platform.

**Key Topics:**
- 👥 **Account & Role Management** — Create accounts, assign roles, manage permissions
- 🏗️ **Product Development** — Category management, standard functions, five-tuple generation
- 📱 **APP Management** — Create apps, configure settings, manage publishing
- 🛠️ **Customer Relationship** — Knowledge base, after-sales, push notifications
- 💾 **Cloud Storage** — Plan configuration, order management
- 🔧 **Device Management** — Unbind devices, batch operations

[Go to Admin Manual →](./admin/)

---

### 📗 Developer Manual
For hardware developers integrating devices with the DMP platform.

**Key Topics:**
- 🔨 **Product Development** — Create products, configure functions
- 📦 **Firmware Management** — Repository management, OTA upgrades
- 🖼️ **Device Interaction** — Product display, functional grouping
- ❓ **FAQ** — Common questions and troubleshooting

[Go to Developer Manual →](./developer/)

---

Choose your role to get started:

| Role | Start Here | Description |
