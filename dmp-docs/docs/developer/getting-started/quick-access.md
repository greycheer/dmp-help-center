# Quick Access to DMP Platform

## DMP Platform Domain

Please contact the platform team for the DMP Platform Domain URL.

## Account and Permission Application

The DMP platform does not support self-service account creation. Please provide the required information (see [Account Setup](../../admin/quick-start/account-setup)) to DMP Platform Admin to create an account.

## Access Steps

### 1. Obtain Platform URL

Contact the platform team to get the DMP platform domain address.

### 2. Apply for Account

Submit the following to platform admin:
- Company information
- Personal information
- Verification email (overseas domains recommended for ODM environments)

### 3. Receive Credentials

Platform admin will create your account and send login credentials.

### 4. First Login

Access the platform domain and log in with provided credentials.

### 5. Start Development

Navigate to **Product** → Continue Development → Select **Device Development** to begin product configuration.
