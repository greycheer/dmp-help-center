# Developer Introduction

## Purpose of This Guide

This manual provides developers with a comprehensive and systematic operational guide for using the DMP platform. By reading this manual, developers can quickly understand the platform's core capabilities and complete the full development workflow, including:

- Product creation
- Product feature configuration
- Device interaction setup
- Hardware development
- Device debugging

## Platform Overview

The DMP platform is a comprehensive platform serving hardware manufacturers with device management capabilities at its core. Key capabilities for rapid hardware product access include:

- **Multi-category Support**: Hardware products of different categories can connect via Wi-Fi, 4G, LAN, etc.
- **Custom Function Configuration**: Support custom configuration of product functions
- **Custom APP Panels**: Support custom APP/large-screen interactive panels
- **Flexible OTA**: Support flexible OTA and debugging capabilities

## Development Workflow

```
1. Product Creation → 2. Thing Model Definition → 3. Firmware Development → 4. Device Debugging → 5. Production Deployment
```
