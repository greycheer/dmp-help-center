# Account Setup

## Registration Process

### Step 1: Contact Platform Admin

The DMP platform does not support self-service account creation. Please provide the following information to the DMP Platform Admin:

**Company Information:**
- Company Name
- Company Address
- Company Contact Person
- Company Contact Details

**Personal Information:**
- Account Nickname (according to customer requirements; in some ODM environments, Chinese characters or Pinyin may not be allowed in nicknames)
- Verification Email (to receive verification codes)
  - Note: In overseas ODM environments, domestic email domains (e.g., qq.com, 163.com, domestic corporate domains) are not allowed. Overseas domains such as Gmail are recommended.

### Step 2: Wait for Account Creation

The platform admin will create your account and send login credentials.

### Step 3: First Login

After registration is successful, log in to the console using your email and password:

```
Login URL: [Contact platform team for DMP Platform Domain]
```

## Initial Configuration

After first login, it is recommended to complete the following configurations:

- [ ] Complete company information
- [ ] Set up security phone/email
- [ ] Configure notification preferences
- [ ] Create your first project
