# Platform Overview

## What is DMP?

**DMP (Device Management Platform)** is a comprehensive device lifecycle management platform designed for IoT hardware manufacturers and brands.

## Core Capabilities

### 1. Device Access Management
- Support multiple connection methods: Wi-Fi, Cellular (4G), LAN
- One-stop service for device registration, activation, and authentication
- Batch import and automated deployment support

### 2. Real-time Monitoring & Alerts
- Real-time device online status display
- Automatic anomaly alerts and notifications
- Multi-dimensional data visualization

### 3. Remote Operations & Upgrades
- OTA (Over-The-Air) firmware remote upgrades
- Grayscale release strategy control
- Batch upgrade task management

### 4. Value-added Service Operations
- Cloud storage service management
- AI intelligent analysis services
- Subscription billing and invoice management

### 5. Open Ecosystem Integration
- Complete RESTful API suite
- Webhook event push notifications
- SDK for rapid integration

## Use Cases

| Scenario | Description |
|----------|-------------|
| **Smart Home** | Manage smart speakers, cameras, door locks, and other devices |
| **Security Surveillance** | Monitor NVR, IPC, and other device operational status |
| **Industrial IoT** | Manage sensors, gateways, and other industrial equipment |
| **Smart Retail** | Monitor POS machines, electronic shelf labels, and other devices |
