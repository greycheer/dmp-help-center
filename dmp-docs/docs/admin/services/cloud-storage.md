# Cloud Storage Plan Configuration

## Section Overview

The DMP Management Console supports configuring and publishing cloud storage plans. Once configured, these plans will be displayed in the App and available for users to purchase.

## Configuration Process

The cloud storage plan configuration process consists of four steps:

1. **Define cloud storage templates** (configure storage duration, cloud service provider, etc.)
2. **Define cloud storage products** (set pricing and purchase restrictions)
3. **Define cloud storage product groups** (group products for easier management)
4. **Configure cloud storage shelves** (publish product groups to different clients)

## Step 1: Define Cloud Storage Templates

**Navigation Path:** [Marketing] → [Value-added Services] → [Cloud Storage Templates]

### Create Template

1. Click **Create Template**

![Cloud Storage Templates](/img/admin/cloud-storage-templates.png)

2. Define basic template information:
   - Template name
   - Storage duration (e.g., 7 days, 30 days)
   - Cloud service provider
   - Storage specifications

![Create Template](/img/admin/create-template.png)

3. Save the template

## Step 2: Define Cloud Storage Products

**Navigation Path:** [Marketing] → [Value-added Services] → [Mall Management] → [Commodity Management]

### Create Product

1. Click **Create Product**

![Create Product](/img/admin/create-product.png)

2. Select a template as the base
3. Define pricing and details

![Product Details](/img/admin/product-details.png)

4. Save the product

## Step 3: Define Product Groups

**Navigation Path:** [Marketing] → [Value-added Services] → [Mall Management] → [Commodity Grouping]

### Create Product Group

1. Click **Create Group**

![Product Grouping](/img/admin/product-grouping.png)

2. Enter group name and associate multiple products

![Product Group Add](/img/admin/product-group-add.png)

3. Save the group

## Step 4: Configure Product Shelves

**Navigation Path:** [Marketing] → [Value-added Services] → [Mall Management] → [Product Shelves]

### Publish Products

1. Select **Product Group**

![Product Shelves](/img/admin/product-shelves.png)

2. Configure shelf placement and complete publishing
