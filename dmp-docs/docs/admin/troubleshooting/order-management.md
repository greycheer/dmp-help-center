# Order Management

## Cloud Storage Order Management

**Navigation Path:** [Marketing] → [Mall Management] → [Commodity Order]

### Search Orders

Use the search bar at the top to locate orders by:
- Order ID
- User ID
- Product name
- Date range
- Order status

### Order/Service Cancellation

1. Locate the order using search
2. In the corresponding order row, click the **Unsubscribe** button
3. In the pop-up window, select the appropriate:
   - Refund method
   - Service cancellation method
4. Confirm the operation

### Order Status Types

| Status | Description |
|--------|-------------|
| **Active** | Order is valid and service is active |
| **Expired** | Service period has ended |
| **Cancelled** | Order has been cancelled |
| **Refunded** | Refund has been processed |
