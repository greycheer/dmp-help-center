# Knowledge Base Configuration

## Section Overview

The DMP Management Console supports configuring Frequently Asked Questions (FAQs) for both the App and different device models, helping reduce the workload of manual customer support.

## Knowledge Base Types

Based on the scope of the content, the Knowledge Base is divided into:

| Type | Scope | Entry Point |
|------|-------|-------------|
| **App Knowledge Base** | App-level common questions | App settings → Help |
| **Product Knowledge Base** | Device-specific questions | Device settings → Help |

The operation steps and interfaces for both are largely similar.

## Create Knowledge Base

### Step 1: Create Knowledge Base Entry

1. Click **Create Knowledge Base**
2. Select the target **App**
3. Select **Languages** to support
4. Click **Confirm**

**Notes:**
- The associated App cannot be changed after creation
- Languages can still be added or modified after creation

### Step 2: Create First-level Categories

1. Click **Management Q&A** to enter the Q&A Management page
2. Click **Add First-level Category**
3. Enter category name
4. Save

### Step 3: Create Subcategories and Articles

After creating a first-level category, you can either:
- Create second-level categories, or
- Add knowledge articles directly under the first-level category

**Important Notes:**
- Once knowledge articles are added under a first-level category, second-level categories can no longer be created under it
- When adding knowledge articles, make sure to configure multiple languages
- Each language requires clicking **Save** once

### Article Content

Each article should include:
- **Title**: Clear, concise question
- **Content**: Detailed answer with steps if applicable
- **Images**: Screenshots or diagrams (optional but recommended)

## Manage Knowledge Articles

After adding knowledge articles, they can be managed via the operation column:

| Function | Description |
|----------|-------------|
| **Publish** | Publish the article to production environment after editing |
| **Sort** | Customize display order (smaller numbers displayed higher) |
| **Edit** | Modify article content |
| **Delete** | Remove the article |

## Product Knowledge Base

For **Published** products, a Product Knowledge Base is automatically created.

On the Product Knowledge Base page, click **Edit** to manage the corresponding knowledge entries.

## Best Practices

- Organize content hierarchically (Category → Subcategory → Article)
- Use clear, searchable titles
- Include step-by-step instructions
- Add screenshots for complex operations
- Regularly review and update outdated content
